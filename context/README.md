# Private Context (Local Only)

This folder stores private profile materials and should never be pushed to a public repository.

## Intended layout

- `context/private/resume/`: source resume files (`.docx`, `.md`)
- `context/private/profile-drafts/`: draft profile narratives and positioning notes
- `context/private/notes/`: private writing notes and references
- `context/private/tools/`: helper scripts for private document generation

## Rule

`context/*` is ignored by Git (except this README) to prevent accidental publication.
