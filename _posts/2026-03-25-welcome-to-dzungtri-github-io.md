# Welcome to dzungtri.github.io

I consolidated profile content and blog publishing into one repository:

- Source content is in `_posts/`, `_data/`, `_includes/`, and Jekyll layouts
- Build runs with `bundle exec jekyll build`
- Deployment runs from GitHub Actions on push to `main`

This setup keeps the writing archive simple, durable, and close to the old Jekyll workflow.
